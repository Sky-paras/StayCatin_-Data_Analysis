# 🏡 StayCatin Property Analytics Dashboard

## 📌 Project Overview

This project analyzes short-term rental property data to uncover
insights into pricing, earnings, property types, and seasonal trends.
The dashboard provides a comprehensive view of host performance,
regional earnings, and pricing strategies.

------------------------------------------------------------------------

## 🎯 Objectives

-   Analyze pricing and earnings patterns across hosts
-   Identify seasonal trends in property pricing
-   Compare premium vs budget properties
-   Evaluate regional performance
-   Understand property distribution by room type

------------------------------------------------------------------------

## 📊 Dashboard Insights

### 1. Price and Earnings by Host

-   High-earning hosts like **SeaBreeze** and **710 Beach Rentals**
    dominate revenue.
-   Strong positive relationship between total earnings and pricing.

### 2. Seasonal Pricing Trend

-   Prices peak around **July (\~334)** indicating high demand.
-   Lowest pricing observed in **February (\~225)**.
-   Clear seasonality trend → useful for dynamic pricing strategies.

### 3. Room Type Distribution

-   **Entire home/apartment dominates (\~22,977 beds)**.
-   Private rooms are significantly lower → indicates premium
    preference.

### 4. Regional Earnings

-   **Alpine and Escondido** contribute major earnings.
-   Some regions show minimal or zero contribution → opportunity areas.

### 5. Premium Cities

-   Certain cities command significantly higher average prices.
-   Indicates location-driven pricing advantage.

### 6. Expensive vs Budget Properties

-   Luxury property types like **Castle and Condominium** have highest
    prices.
-   Budget options like **Cabin and Apartment** are more affordable.

------------------------------------------------------------------------

## 🛠 Tools Used

-   Tableau (Dashboard creation)
-   Data Visualization Techniques
-   Business Analysis

------------------------------------------------------------------------

## 💡 Key Business Insights

-   Seasonal demand heavily impacts pricing → pricing should be dynamic.
-   Few top hosts drive majority revenue (Pareto principle).
-   Entire homes are most preferred property type.
-   Premium locations significantly increase earnings potential.

------------------------------------------------------------------------

## 🚀 Conclusion

This project demonstrates how data visualization and analysis can drive
pricing strategy, improve revenue, and identify growth opportunities in
the rental property market.
